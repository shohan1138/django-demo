# Django Event Registration App

This project is a simple Django application that allows users to register for events by providing their name, contact details, email, event name, and an optional file upload.

## Features
- Event registration form with validation
- File upload (max size: 10 MB)
- Basic Django setup for learning purposes

## How to Run
```bash
git clone https://github.com/<your-username>/django-demo.git
cd django-demo
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
